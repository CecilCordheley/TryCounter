Comment Installer ?

  1. Copier l'url de la page
  2. Coller la en source navigateur sur votre logiciel StreamLabs ou Obs
  3. Taille optimale [largeur : 256 /hauteur : 77]
  4. Cliquer sur intéragir
  5. [+] du clavier pour incrémenter
  6. [0] du clavier pour remettre à 0
  7. [Escape] du clavier pour revenir à la valeur de base
